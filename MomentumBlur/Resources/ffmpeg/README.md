# FFmpeg binaries

FFmpeg full build binaries are not committed to this public source repository because GitHub blocks files larger than 100 MB.

The official release ZIP uploaded in GitHub Releases includes the required bundled FFmpeg files:

- ffmpeg.exe
- ffprobe.exe

For local source builds, download the FFmpeg 8.1.1 full Windows build and place it under:

MomentumBlur/Resources/ffmpeg/ffmpeg-8.1.1-full_build/bin/
